Documentación básica de Docker

Resumen de los comandos y conceptos esenciales para trabajar con Docker: búsqueda, gestión de imágenes y contenedores, y creación de imágenes personalizadas.

Gestión de imágenes y contenedores

  • Buscar una imagen: 
    docker search nombre_imagen
    Ejemplo para una versión específica: docker search ubuntu:14.04
  • Descargar una imagen: 
    docker pull nombre_imagen
  • Listar imágenes descargadas: 
    docker images
  • Ver todos los contenedores (incluidos los detenidos): 
    docker ps -a
  • Ver solo los contenedores en ejecución: 
    docker ps
  • Crear un contenedor: 
    docker run nombre_imagen
    Puedes referenciar un contenedor por las tres primeras letras de su ID.
  • Ejecutar bash en un contenedor Ubuntu: 
    docker run -i -t ubuntu bash
    -i y -t permiten modo interactivo.
  • Iniciar un contenedor detenido: 
    docker start idContenedor
  • Conectar al bash de un contenedor: 
    docker attach idContenedor
  • Asignar nombre al contenedor al crearlo: 
    docker run --name MiContenedor -it nombre_imagen
  • Salir del contenedor sin detenerlo: 
    Ctrl + p + q
  • Detener un contenedor: 
    docker stop idContenedor
  • Actualizar dependencias dentro del contenedor: 
    apt-get update
    Algunas imágenes pueden no soportar todos los comandos.
  • Instalar paquetes: 
    apt-get install paquete

Creación y gestión de imágenes personalizadas

  • Crear una imagen a partir de un contenedor: 
    docker commit idContenedor nombreImagen
    Ejemplo: docker commit 123 NuevaImagen
  • Crear imagen con configuración específica: 
    docker commit --change='CMD ["apache2ctl", "-D", "FOREGROUND"]' -c "EXPOSE 85" idContenedor nombreImagen
  • Crear y exponer puertos en un nuevo contenedor: 
    docker run -d -p 5000:80 nombreImagen
    Accede a localhost:5000 para comprobar el servicio.
    Para cambiar el puerto: docker run -d -p 5001:80 nombreImagen

Uso de Dockerfile

  • Crear un Dockerfile: 
    nano Dockerfile
    O usa gedit si prefieres entorno gráfico.
  • Construir una imagen desde un Dockerfile: 
    docker build -t nombreImagen /ruta/de/la/carpeta
  • Ejecutar la imagen y asignar puerto: 
    docker run -d -p 85:80 nombreImagen
  • Comprobar contenedores en ejecución: 
    docker ps
    Accede a localhost:85 para visualizar el servicio.
